## Keybindings
Default i3 but some added

alt + ctrl + g = brave <br/>
alt + ctrl + k = keepassxc <br/>
alt + ctrl + p = pavucontrol <br/>
alt + ctrl + l = lockscreen <br/>
alt + ctrl + d = dockterm <br/>
alt + ctrl + s = new nvim Window <br/>
alt + b = background <br/>
alt + m = screen off <br/>
print = flameshot <br/>
